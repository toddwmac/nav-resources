ℹ️ About This Collection

What You'll Find Here

This directory contains a comprehensive database of navigation resources compiled specifically for Western US overland travel. Whether you're piloting a 12-foot motorhome or riding adventure bikes through BDR routes, these tools help you navigate safely, legally, and confidently.

Vehicle Profile

Primary Rig: 12-foot height motorhome + adventure motorcycle carry
Region: Western United States (AZ, CA, CO, ID, MT, NV, NM, OR, UT, WA, WY)
Key Constraint: Low clearance avoidance is critical

Resource Categories

  • RV Navigation: Low-clearance databases, truck routing, dedicated GPS hardware
  • Adventure Motorcycle: BDR routes, off-road mapping, GPX track management
  • GeoPDF & Agency Maps: USFS MVUMs, BLM land boundaries, official forest service maps
  • Camping & Boondocking: Dispersed camping finders, federal campground reservations
  • Print Atlases: Essential backup navigation for when electronics fail

⚙️ How It Works

📄

JSON Data Files

Structured resource databases with metadata including URLs, categories, cost tiers, offline capability, and priority levels. Import these into the HTML apps or use them as reference.

🖥️

Interactive HTML Apps

Single-file web applications with full CRUD operations. Filter, sort, search, and manage resources. Each app uses localStorage for persistence.

💬

User Feedback System

Add your own commentary, notes, and experiences to each resource. All feedback is stored locally and can be exported for backup or sharing.

🌙

Theme System

Switch between light and dark modes. Your preference is saved automatically. The interface adapts seamlessly for day and night use.
